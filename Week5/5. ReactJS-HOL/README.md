# Cohort Details App

## Installation

```bash
npm install
npm start
```

## Project Description
This project demonstrates styling with a CSS module and conditional heading colors based on the cohort status.

## Folder Structure
- src/App.js
- src/CohortDetails.module.css
- src/index.js

## Features Implemented
- CSS module styling
- Conditional heading color for ongoing vs non-ongoing cohorts
- Clean React 18 functional component implementation
