# Ticket Booking App

## Installation

```bash
npm install
npm start
```

## Project Description
This project demonstrates a simple login/logout flow with conditional rendering in React.

## Folder Structure
- src/App.js
- src/index.js

## Features Implemented
- Guest page showing flight details and restricted booking
- `Login` button updates state to render the user page
- `Logout` button returns to guest state
- React state-driven conditional rendering
