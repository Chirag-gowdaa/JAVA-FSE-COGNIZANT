# Cricket App

## Installation

```bash
npm install
npm start
```

## Project Description
This project demonstrates list rendering, filtering, array destructuring, spread operations, and conditional rendering in React.

## Folder Structure
- src/App.js
- src/index.js

## Features Implemented
- Array of 11 player objects
- `map()` to render players
- `filter()` with arrow functions for score < 70
- `IndianPlayers` component using destructuring
- Split into odd and even teams
- Merged players using spread operator
- Conditional rendering via `flag`
