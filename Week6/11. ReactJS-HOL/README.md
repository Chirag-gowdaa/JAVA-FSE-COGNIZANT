# Event Examples App

## Installation

```bash
npm install
npm start
```

## Project Description
This project demonstrates React event handling with a counter, welcome alert, synthetic event button, and INR to Euro conversion.

## Folder Structure
- src/App.js
- src/index.js

## Features Implemented
- Counter with Increment and Decrement buttons
- Multiple method invocation from one click
- `Say Welcome` button with argument passing
- `OnPress` button using synthetic event
- Currency conversion with constant exchange rate
