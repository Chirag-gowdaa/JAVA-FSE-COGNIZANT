# Office Space Rental App

## Installation

```bash
npm install
npm start
```

## Project Description
This project renders an office rental page using JSX, inline styling, and a mapped office list.

## Folder Structure
- src/App.js
- src/index.js

## Features Implemented
- JSX page layout with heading and office image
- Office object list with `name`, `rent`, and `address`
- `map()` rendering of all office entries
- Rent color logic using inline styles
