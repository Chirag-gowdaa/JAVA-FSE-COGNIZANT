# Blogger App

## Installation

```bash
npm install
npm start
```

## Project Description
This project showcases multiple React conditional rendering techniques using reusable components and mapped arrays.

## Folder Structure
- src/App.js
- src/index.js

## Features Implemented
- `BookDetails` using `if/else`
- `BlogDetails` using ternary operator
- `CourseDetails` using logical `&&`
- Element variables for rendering lists
- Reusable, keyed components with array mapping
